# Linux安装高版本gcc,g++
## 添加相应的源
```bash
sudo add-apt-repository ppa:ubuntu-toolchain-r/test
```
## 更新软件源
```bash
sudo apt-get update
```
## 安装
```bash
sudo apt-get install gcc-11 g++-11
```
